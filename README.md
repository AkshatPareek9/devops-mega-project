## Tech stack used in this project:

1. Github (code clone)
2. ReactJS (Frontend)
3. NodeJS (Backend)
4. MongoDB (Database)
5. RedisDB (Caching)
6. Terraform (Infrastructure)
7. AWS (Cloud)
8. Jenkins (CI)
9. Docker (containerization)
10. OWASP (dependency check)
11. SonarQube (Quality check)
12. Trivy (File system scan)
13. ArgoCD (CD)
14. AWS EKS (Kubernetes)
15. Helm (Packaging)
16. Prometheus and Grafana (Monitoring and Visualization)
